# Developer Portfolio

A modern portfolio website built with Next.js App Router, React, Tailwind CSS, Framer Motion, and React Three Fiber.

## Features

- Deep dark UI with neon glassmorphism
- Interactive 3D hero background
- Infinite animated skills marquee
- Bento-style projects section
- Animated contact form with client-side validation

## Scripts

- `npm run dev` — start the development server
- `npm run build` — create the production build
- `npm run start` — run the production server
- `npm run lint` — run ESLint
